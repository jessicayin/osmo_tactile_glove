# PCB Files

This directory contains all printed circuit board design files.

## Structure

- `main-board/` - Primary control board design files
- `sensor-board/` - Tactile sensor interface board design files
- `schematics/` - Circuit diagrams and schematics

## File Formats

Common formats include:
- Gerber files (.gbr, .ger)
- KiCad project files (.kicad_pcb, .kicad_sch)
- Eagle files (.brd, .sch)
- PDF schematics
