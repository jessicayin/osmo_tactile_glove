# README

This folder contains step-by-step instructions for assembling the OSMO glove hardware.

## Files

Please review the files in the following order:

1. `bom.md` - Bill of materials
2. `sensor_assembly.md` - How to wire and prepare the sensor + MCU boards
3. `flash_firmware.md` - How to flash the firmware to a new glove
4. `glove_assembly.md` - How to assemble the glove
5. `troubleshooting.md` - Common issues and solutions
