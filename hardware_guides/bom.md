# Bill of Materials

This is for making 1 x OSMO tactile glove. 

### Sensor Assembly

#### Electronics
- 1 MCU board | Glove microcontroller
- 12 sensor boards: 5 x Variant A, 5 x Variant B, 2 x Variant C | Tactile + IMU sensor boards
- Tinsel wire | Connect sensors to MCU board
- 1 x MuMetal sheet | Crosstalk reduction
- Silpoxy silicone adhesive | Glue sensor assembly together

#### Magnetic Elastomers
- Magnequench magnetic particles
- Smooth-On 00-30 Platinum Elastomer

- Pulse magnetizer
Refer to reskin / anyskin website for more details / videos


### Equipment
- Soldering iron
- Solder
- Plastic cups 
- Cotton q tips
- 3D printed molds (link)
- Demolder
- Multimeter
- Sandpaper
- (Optional, recommended) Chipquik flux
- (Optional, recommended) UV glue + UV flashlight
- (Optional, recommended) Xacto knife

### Glove Assembly
